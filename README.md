# Coach AJ Health & Wellness Website

A comprehensive Next.js website for Coach AJ's holistic health coaching practice, featuring AO Scan Technology, personalized health evaluations, and the Mighty 90 essential nutrients philosophy.

## Features

- **AO Scan Technology Information**: Detailed pages about frequency-based assessment and all scan modules
- **Personalized Health Evaluations**: Integration with MyHealthEvaluation.com
- **Philosophy & Education**: Complete Triangle of Disease framework and Mighty 90 nutrients education
- **SEO Optimized**: Comprehensive LSI keywords and metadata for top search rankings
- **Responsive Design**: Mobile-first approach using TailwindCSS
- **Social Media Integration**: Links to all Daily with Doc platforms and Coach AJ resources
- **Product Resources**: Integrated links to 8+ product information websites

## Technology Stack

- **Framework**: Next.js 14 (React 18)
- **Styling**: TailwindCSS
- **Icons**: Lucide React
- **Language**: TypeScript
- **Deployment**: Netlify-ready

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in your browser.

### Build

```bash
npm run build
```

### Production

```bash
npm start
```
